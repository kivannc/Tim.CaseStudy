# Tim.CaseStudy



Clone down this repository. You will need `net7` , `node` and `npm` installed globally on your machine.  


Change Diretory:

`cd Library.API`

Installation api: 

`dotnet restore`

To Start API:

`dotnet run`

Visit Swagger Page:

`https://localhost:7133/swagger/index.html`

Change Directory:

`cd library-client`

Installation web app:

`npm install`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
